# Pravalika Kuchipudi Kalakshetram

A premium cultural institution website dedicated to preserving, promoting, and teaching the rich heritage of Kuchipudi, one of India's most celebrated classical dance forms.

This website serves as the official digital presence of **Pravalika Kuchipudi Kalakshetram**, providing information about the academy, its history, courses, performances, gallery, and contact details.

## Installation

Clone the repository:

```bash
git clone https://github.com/OceanFlowhq/Pravalika-Kalakshetram.git
```

Navigate into the project:

```bash
cd Pravalika-Kalakshetram
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
